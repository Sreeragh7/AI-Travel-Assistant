# Travel Itinerary Assistant

A web app that generates a personalized travel itinerary based on your destination, budget, and number of days. Features include:

- Day-by-day itinerary with breakfast, places to visit, lunch, dinner, and accommodation
- Download itinerary as PDF
- Interactive map with search for each day's places

## Features
- Enter destination, budget, and days
- Get a detailed itinerary for each day
- Download as PDF
- View and search places on a live map

## Setup
1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the app:
   ```
   python app.py
   ```
4. Open your browser at `http://localhost:5000`

## Technologies Used
- Python, Flask
- Leaflet.js, OpenStreetMap
- xhtml2pdf
- Bootstrap
